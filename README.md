# 🌍 WanderNest – Travel Agency Website

A modern and responsive travel agency website built using HTML and CSS. WanderNest helps users explore popular travel destinations through a clean, attractive, and user-friendly interface.

## ✨ Features

- 🌍 Responsive design
- ✈️ Travel destination search
- 🏖️ Popular destinations section
- 📅 Booking form
- 👥 About section
- ⭐ Customer reviews
- 📞 Contact section

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```
WanderNest/
│── index.html
│── style.css
│── README.md
```

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/Harsh-12saini/WanderNest.git
```

2. Open the project folder.

3. Open `index.html` in your web browser.
